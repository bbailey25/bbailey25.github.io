Image Meta Save

Description

Image Meta Save can be used to add meta data to images and upload them to the WordPress media library. The meta data can be saved together as a preset in a database to be used over and over.

The fields supported by this plugin are title, alternative text, caption, description, latitude, and longitude.

All the fields can be seen in the media library, with the exception of latitude and longitude. Those can be seen in the wp_postmeta database.
